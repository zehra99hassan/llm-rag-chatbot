# LLM RAG Chatbot

An AI-powered chatbot system built using Retrieval-Augmented Generation (RAG), vector search, and LLM workflows.

This project demonstrates how modern AI systems can retrieve contextual information, generate intelligent responses, and manage conversational workflows efficiently.

---

## Features

- Retrieval-Augmented Generation (RAG)
- Semantic Search using Embeddings
- Context-aware AI responses
- FastAPI backend integration
- Vector database workflow
- AI summarization pipeline
- Modular chatbot architecture

---

## Tech Stack

- Python
- FastAPI
- Qdrant
- Docker
- LLM APIs
- Embeddings
- REST APIs

---

## System Workflow

1. User sends a query
2. Query converted into embeddings
3. Relevant context retrieved from vector database
4. LLM generates contextual response
5. Response returned through API layer
